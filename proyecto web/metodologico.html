Tipos de Metodologías

Responsive Web Design (RWD)

Responsive Web Design (RWD)

Esta metodología se centra en crear páginas web que se adapten y se vean bien en diferentes dispositivos y tamaños de pantalla, como computadoras de escritorio, tabletas y teléfonos móviles. Se utilizan técnicas como el diseño fluido y los media queries para lograr este objetivo.

Mobile-First Design

Esta metodología implica diseñar primero la versión móvil de una página web y luego adaptarla

Más sobre Mobile-First Design

Esta metodología implica diseñar primero la versión móvil de una página web y luego adaptarla para dispositivos más grandes, como computadoras de escritorio. Esto garantiza que la experiencia del usuario en dispositivos móviles sea óptima, ya que cada vez más personas acceden a Internet a través de sus teléfonos.

Agile Development

El desarrollo ágil es un enfoque iterativo e incremental para el desarrollo de software

Más sobre Agile Development

El desarrollo ágil es un enfoque iterativo e incremental para el desarrollo de software, que se puede aplicar también al desarrollo de páginas web. Se divide el proyecto en ciclos de desarrollo cortos llamados "sprints", durante los cuales se entregan incrementos funcionales del producto.

Tipo de Página Web

Sección Contenido

Página web dinámica

Las páginas web dinámicas muestran contenido que puede cambiar según la interacción del usuario o la información almacenada en una base de datos. Estas páginas suelen estar construidas con lenguajes de programación del lado del servidor, como PHP, Python o Ruby on Rails, y pueden incluir características como formularios interactivos, inicio de sesión de usuario y actualización en tiempo real.

Una página web dinámica permite actualizar el contenido de manera rápida y sencilla. Dado que la información en una carrera como la ingeniería de sistemas está en constante evolución con los avances tecnológicos y cambios en el campo, es crucial poder actualizar rápidamente el contenido del sitio para mantenerlo relevante y preciso; nos permite agregar, editar y eliminar contenido de manera eficiente sin necesidad de conocimientos técnicos avanzados. Esto facilita la tarea de mantener el sitio actualizado y en funcionamiento sin problemas.

Incluye interactivas que mejoran la experiencia del usuario, como formularios de contacto, evaluaciones en línea, foros de discusión y sistemas de comentarios. Estas características pueden fomentar la participación de los estudiantes y promover el aprendizaje colaborativo, además, puede adaptarse para ofrecer contenido personalizado según las preferencias y necesidades individuales de los estudiantes del Colegio León de Greiff de los grados decimo y once.

Encuesta

Una encuesta es un instrumento de recolección de información que consiste en una serie de preguntas estructuradas diseñadas para recopilar datos sobre opiniones, actitudes o características de una población específica. Se utiliza para obtener información cuantitativa o cualitativa sobre un tema en particular, y puede ser administrada de manera presencial, telefónica, en línea o a través de otros medios.

Instrumento Más Adecuado

Contenido

Se implementará como instrumento de recolección la encuesta ya que es versátil, por este medio se podrá llegar a una cuantificación de las personas y sus necesidades para tener definido el público al que se dirigirá la página para lograr resultados exitosos con la misma. Por medio de los resultados que arroje la encuesta se tendrá en cuenta para la constante actualización de la página y sus usuarios personalizados.
