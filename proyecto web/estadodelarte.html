ESTADO DEL ARTE



Proporciona una revisión exhaustiva y actualizada del conocimiento existente sobre un tema específico, ofreciendo contexto, identificando lagunas en la investigación y estableciendo la credibilidad del contenido. Actúa como una base sólida que educa a los lectores sobre los desarrollos, teorías y tendencias más relevantes, al mismo tiempo que guía futuras investigaciones y mantiene la relevancia del sitio en su campo de especialización.

SOFTWARES


Los softwares que se van a utilizar para llevar a cabo este proyecto serán:

  • Librería Bootstrap V5.3.3: Para facilitar la programación.
  • HTML 5 Y CSS 3: Como lenguajes de programación.
  • Visual Studio Code: Como compilador o editor de código.

REQUERIMIENTOS HARDWARE

Un procesador de doble núcleo o superior es suficiente.

Al menos 2 GB de RAM, aunque más es preferible para un rendimiento óptimo

Espacio suficiente en disco duro para el sistema operativo y las herramientas de desarrollo

COLORES Y PSICOLOGIA DEL COLOR

Se debe crear una página neutral y elegante; se usará la psicóloga del color para poder generar una atracción a los usuarios mediante las emociones que estos transmitirán
  • Azul: Transmite confianza, seguridad y tecnología.
  • Verde: Representa crecimiento, innovación y equilibrio.
  • Gris: Sugiere profesionalismo, solidez y formalidad.
  • Blanco: Simboliza limpieza, claridad y simplicidad.
  • Negro: Aporta elegancia, modernidad y poder.

FONDOS Y PSICOLOGIA DEL COLOR

Negro

Utilizado como fondo de página, el negro transmite autoridad, elegancia y poder. Este color

Gris oscuro

Agregar gris oscuro al fondo de una página muestra seriedad, formalidad y estabilidad. Este color transmite confianza y profesionalismo, lo que puede atraer a aquellos interesados en carreras técnicas y estructurales como la ingeniería de sistemas. Además, el gris oscuro aporta profundidad al diseño sin restar elegancia y crea un ambiente sofisticado y atractivo para los visitantes.
